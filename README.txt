MasterKoleksiyon - PWA Paket

✅ Bu paket PWA (kurulabilir uygulama) olarak çalışır.

ÖNEMLİ:
- Service Worker, "file://" ile çalışmaz. Uygulama HTTP/HTTPS üzerinden açılmalı.

PC'de hızlı çalıştırma:
1) Bu klasörü açın.
2) Terminal/CMD:
   python -m http.server 8080
3) Tarayıcı: http://localhost:8080
4) Chrome/Edge: MenÜ > Uygulamayı yükle / Ana ekrana ekle.

Android:
- Dosyaları bir HTTPS barındırmaya koyup (GitHub Pages, Netlify vb.) linki Chrome'da açın.
- Sağ üst menü > Ana ekrana ekle.

Not:
- Offline çalışır; ancak Google Fonts internet yokken yüklenmeyebilir. Uygulamanın kendisi çalışır.
